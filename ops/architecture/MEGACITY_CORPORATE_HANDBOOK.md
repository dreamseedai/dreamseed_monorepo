# 📌 0. 서문 (Introduction)

DreamSeedAI Corporate Handbook은 DreamSeedAI 구성원이 **어떻게 일하고, 소통하고, 배우고, 성장하고, 협력하는지에 대한 공식 기준 문서**입니다.

DreamSeedAI는 단순 교육회사가 아니라 **AI 기반 글로벌 MegaCity를 건설하는 조직**이기 때문에,
강력한 문화·원칙·조직 운영 체계가 필수입니다.

이 문서는 다음 내용을 포함합니다:

# 🧭 2. DreamSeedAI Work Culture

## 2.1 Document-First

* 모든 결정은 문서로 남긴다.
* Slack 대화보다 문서 링크가 기본.

## 2.2 Engineering-Driven

* 기술팀의 의견과 데이터 기반 의사결정이 우선.
* 코드·모델·인프라 품질이 회사 성장의 핵심.

## 2.3 Zero-Drama·Zero-Politics

* 정치/소문/뒷담화 금지.
* 갈등은 사실/데이터로 해결.

## 2.4 Async-friendly

* 시간대가 달라도 협업되도록 문서 기반.

## 2.5 High Ownership

* 불편함을 발견하면 즉시 문서화 → 해결.

# 🕒 3. Employment & Work Policies

## 3.1 근무시간

* 기본은 **자율 근무제**.
* 단, 팀별 Core Time(중복 시간) 3시간 권장.

## 3.2 휴가 정책

```
연차: 연 20일
병가: 별도 제공
리프레시 휴가: 근속 3년마다 10일 추가
```

## 3.3 휴가 원칙

* 무조건 승인 (단, 팀 운영만 영향 없게 조율)
* 휴가 guilt-free

## 3.4 평가 주기

* 반기(6개월) 단위 평가
* 성과 + 성장 + 문서 기여도를 함께 평가

# 🧑‍💼 5. Leadership Principles

DreamSeedAI 리더는 다음을 수행해야 합니다:

```
1) 문서로 명확히 설명한다.
2) Conflict → 데이터 기반 해결.
3) 팀의 생산성을 높인다.
4) 후배/신입을 성장시키는 환경을 만든다.
5) AI Safety와 Privacy를 최우선으로 한다.
6) MegaCity 전체 관점에서 사고한다.
```

# 🛠️ 7. Operating Manuals (팀 운영 기준)


# 🔐 8. Security · Privacy · Compliance

DreamSeedAI는 **AI 교육 플랫폼으로서 가장 높은 보안/프라이버시 기준**을 가지고 있습니다.

### 8.1 보안 원칙

* Zero Trust
* Principle of Least Privilege
* 2FA 필수
* Secrets 무조건 Vault 관리

### 8.2 PII 보호

```
얼굴/음성 데이터 → 7일 후 삭제
Exam Data → 3년 보관
User Data → 가명처리 후 분석
```

### 8.3 규제 준수

* GDPR
* PIPA(한국)
* COPPA(US)
* FERPA(교육 기록)
