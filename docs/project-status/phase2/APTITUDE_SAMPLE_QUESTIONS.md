## Scoring Algorithm

### Step 1: Calculate Dimension Scores

For each dimension:
```
dimension_score = SUM(raw_values) / COUNT(questions)
```

Example:
- Student answers STEM questions: [2, 1, 2, 1, 2]
- STEM_interest = (2 + 1 + 2 + 1 + 2) / 5 = **1.6**

Range: -2.0 to +2.0 (normalized to -1.0 to +1.0 for profile)

---

### Step 2: Calculate Career Fit Scores

**Engineering Fit:**
```
Engineering = 0.40 Ã— STEM + 0.30 Ã— Logical + 0.20 Ã— Practical - 0.10 Ã— Social
```

**Computer Science Fit:**
```
CS = 0.50 Ã— STEM + 0.40 Ã— Logical + 0.10 Ã— Practical
```

**Business Fit:**
```
Business = 0.40 Ã— Verbal + 0.30 Ã— Social + 0.20 Ã— Logical + 0.10 Ã— Practical
```

**Humanities Fit:**
```
Humanities = 0.50 Ã— Verbal + 0.30 Ã— Artistic + 0.20 Ã— Social - 0.20 Ã— STEM
```

**Arts Fit:**
```
Arts = 0.60 Ã— Artistic + 0.30 Ã— Verbal + 0.10 Ã— Social
```

**Medicine Fit:**
```
Medicine = 0.30 Ã— STEM + 0.25 Ã— Logical + 0.25 Ã— Practical + 0.20 Ã— Social
```

**Natural Sciences Fit:**
```
Science = 0.50 Ã— STEM + 0.40 Ã— Logical + 0.10 Ã— Verbal
```
